Описание структуры
NovaHaus/ — корневая папка проекта Django.

settings.py — настройки проекта (база данных, статические файлы, шаблоны).

urls.py — маршруты для всего проекта.

wsgi.py и asgi.py — конфигурация для запуска сервера.

main/ — основное приложение.

models.py — модели базы данных (например, услуги, отзывы, проекты).

views.py — логика представлений (обработка запросов).

templates/ — шаблоны для отображения страниц.

static/ — статические файлы (CSS, JS, изображения).

static/ — папка для статических файлов.

css/ — стили для всех элементов сайта.

js/ — JavaScript-файлы для функционала (калькулятор, чат-бот, анимации).

images/ — изображения для сайта (логотип, слайдер, портфолио).

media/ — папка для медиафайлов, загружаемых пользователями (например, фото проектов).

requirements.txt — список зависимостей (например, Django, библиотеки для AI).

db.sqlite3 — база данных SQLite (используется по умолчанию в Django).
